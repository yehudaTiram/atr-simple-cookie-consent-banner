תוסף קל וגמיש להצגת באנר הסכמה לעוגיות, המותאם במיוחד לאתרים בישראל (תיקון 13 לחוק הגנת הפרטיות). כולל ניהול הסכמה, אינטגרציה עם WooCommerce ואינטגרציית טפסים.

== Description ==

התוסף ATR Simple Cookie Consent Banner נבנה במיוחד עבור אתרים בישראל כדי לסייע לבעלי אתרים להתמודד עם דרישות תיקון 13 לחוק הגנת הפרטיות. התוסף מציג באנר הסכמה עם קטגוריות (חיוני, אנליטיקס, שיווק), מוסיף קבלת מדיניות פרטיות בעמוד התשלום של WooCommerce, ויכול להוסיף תיבת סימון מדיניות פרטיות לטפסים באתר כולו (כולל בוני עמודים פופולריים).

תכונות עיקריות:

- באנר הסכמה חכם עם קטגוריות
- תיבת קבלת מדיניות פרטיות בעמוד התשלום של WooCommerce
- אינטגרציית טפסים גלובלית: הוספת תיבת סימון מדיניות פרטיות לרוב הטפסים באתר
- חלון העדפות והסכמה מפורטת
- אפשרות לחסימת סקריפטי מעקב
- התאמה אישית של טקסטים ועיצוב
- תמיכה מובנית בעברית ובאנגלית (מוכן לתרגום)

הערות חשובות ואחריות:

- התוסף הוא כלי טכני בלבד ואינו מהווה ייעוץ משפטי. השימוש בו אינו מבטיח עמידה בחוק כלשהו (לרבות חוק הגנת הפרטיות בישראל או GDPR). יש להיוועץ ביועץ משפטי בהתאם לצרכים הספציפיים שלכם.
- באחריותכם בלבד. אין אחריות לנזקים מכל סוג.

= Internationalization (i18n) =

- Text domain: `atr-simple-cookie-consent-banner`
- Domain path: `/languages`
- תרגום לעברית כלול. ניתן להוסיף שפות נוספות באמצעות קבצי PO/MO בתיקיית `languages/`.

== Installation ==

1. העלו את קובץ ה-ZIP דרך WordPress → Plugins → Add New → Upload Plugin, או חלצו ל-`wp-content/plugins/`.
2. הפעילו את התוסף.
3. היכנסו ל-Settings → Cookie Consent Banner כדי להגדיר טקסטים, התנהגות ועיצוב.
4. אם אתם משתמשים ב-WooCommerce, ודאו שתיבת קבלת מדיניות הפרטיות מופיעה ופועלת בקופה.

== Frequently Asked Questions ==

= הבאנר לא מופיע =
- ודאו שהתוסף פעיל.
- פתחו את האתר בחלון פרטי/אינקוגניטו.
- נקו קאש באתר/שרת.
- ודאו שהתבנית מפעילה `wp_head()` ו-`wp_footer()`.

= איך משנים טקסטים ועיצוב? =
- דרך Settings → Cookie Consent Banner. ניתן להוסיף CSS למחלקות `.atr-scb-*`. קבצי ה-CSS/JS נטענים מ-`public/atr-scb.css` ו-`public/atr-scb.js`.

= האם יש אינטגרציה עם WooCommerce? =
- כן. ניתן להוסיף תיבת קבלת מדיניות פרטיות לקופה, בהתאם להגדרות האינטגרציה של התוסף.

= האם התוסף מבטיח עמידה בחוק? =
- לא. זהו כלי טכני בלבד. התאימות תלויה באתר, בתהליכי עיבוד המידע ובהנחיית יועץ משפטי.

= האם יש תמיכה בעברית? =
- כן. עברית ואנגלית כלולות. ניתן להוסיף שפות נוספות באמצעות קבצי PO/MO בתיקיית `languages/`.
